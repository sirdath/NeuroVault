# NeuroVault Core Covenant

NeuroVault is being built as a trustworthy open-source memory system, not as a product that takes ownership of a person's memory. The public core follows these commitments:

1. **Works locally without an account.** The core app, local server, indexing, search, automatic context, review, and correction remain usable without signing in to a NeuroVault service.
2. **Files remain yours.** User-authored notes stay readable as ordinary Markdown. Local indexes and derived metadata are rebuildable from documented formats.
3. **No remote kill switch.** An account, license server, organizational change, or unavailable NeuroVault website will not disable an already installed local app or delete local data.
4. **Memory data is not for sale or model training.** NeuroVault does not sell vault data or use it to train a model. A connected AI provider receives only the context disclosed by that connection's data-flow description.
5. **Privacy changes are visible.** A release that changes what can leave the Mac must include a plain-language privacy diff before publication.
6. **Ownership and safety stay in the core.** Export, local backup, deletion, restore, provenance, correction, private controls, and security fixes will not be withheld behind an account or paid tier.

This covenant describes the official NeuroVault project and releases. The open-source license continues to govern the legal permissions granted to the code.
