# NeuroVault Core Covenant

The open-source memory engine now lives in the public
[NeuroVault Core](https://github.com/sirdath/neurovault-core) repository. Its
MIT license and this covenant apply to Core, not to the commercial Desktop UI.

1. **Core works locally without an account.** The memory engine, Markdown
   storage, indexing, local retrieval, HTTP API, MCP bridge, and compatible
   automatic-context hooks remain usable without a NeuroVault account.
2. **Files remain yours.** User-authored notes remain ordinary Markdown.
   Local indexes and derived metadata are rebuildable from documented formats.
3. **No remote kill switch in Core.** An unavailable account, license server,
   or NeuroVault website does not disable an installed Core engine or delete
   local data.
4. **Memory data is not for sale or model training.** NeuroVault does not sell
   vault data or use it to train a model. A connected AI provider receives only
   context disclosed by that connection's data-flow description.
5. **Privacy changes are visible.** A release that changes what can leave the
   device must include a plain-language privacy update.
6. **Core safety stays open.** Exportable Markdown, provenance, correction,
   deletion, and security fixes to the open engine are not withheld behind a
   Desktop purchase.

NeuroVault Desktop is a paid consumer product built on Core. The purchase pays
for the signed application, Memories and Graph interfaces, Review experience,
themes, guided integrations, service management, Store distribution, and
support. Core remains independently useful.

Source code published through NeuroVault Desktop v0.6.0 remains available to
recipients under its historical MIT grant. See
`LICENSES/NeuroVault-v0.6.0-MIT.txt`.
