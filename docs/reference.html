Three retrievers, one fused ranking. Vector, lexical, and graph signals each catch what the others miss; RRF blends them and the cross-encoder sharpens the top.
Every stage is on-device. Embedding, fusion, and reranking all run locally through fastembed ONNX. No query ever leaves the machine.

07

On-device Inference

no per-call model API
100%LOCAL

Embedding and reranking inference run locally. The Direct flavor downloads model files from Hugging Face on first use; the Store bundles its embedding model and excludes the reranker.

Connected AI clients can still send selected context to their configured provider. See PRIVACY.md for the exact boundary.
