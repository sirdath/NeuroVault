Features

Memory that actually stays yours

Everything runs on your laptop. No cloud account, no telemetry, no phone-home. Just markdown files and a SQLite index you can delete at any time.

🔒

Truly local-first

Binds to 127.0.0.1. Your data never leaves your machine. Files are plain markdown you can open in Obsidian at any time.

🧠

Ebbinghaus decay

Memory that forgets on purpose. Strength decays with time, reinforced by recall. The only AI memory app with a biological forgetting curve.

🕸️

Force-directed graph

See how your notes connect. Typed wikilinks — manual, uses, contradicts — render as differently-colored edges.

⚡

Hybrid search

Semantic embeddings + BM25 keyword + graph traversal, fused via RRF. Cross-encoder reranker as optional second pass.

🔌

MCP-native

✓

Every AI rewrite of your wiki is diffable, justified, and sign-off gated. No competitor ships this — you keep control.

How it works

Three layers, one brain

Inspired by Karpathy's LLM wiki pattern. Raw sources never change. Compiled wiki pages get rewritten (with diffs you approve). Your conventions live in the schema.

1

Capture anything

Drop markdown into a folder, paste from anywhere, or let agents write through MCP. Raw sources are append-only — nothing ever gets lost.

2

Compile into wiki

Claude synthesizes your raw notes into canonical wiki pages with Timeline footers. Every rewrite is a diff you can approve, reject, or edit.

3

Recall instantly

Cmd+K fuzzy search, semantic recall, or ask Claude directly — it calls the MCP recall tool and threads the answer back in context.

🛡️

Your data never leaves your machine

NeuroVault is the only AI memory product that can credibly say "100% local" and back it up. No account, no cloud, no analytics, no telemetry. If you delete ~/.neurovault, your brain is gone for good. Read the privacy manifesto →

Install

Two minutes to your first memory

Download the app, open it, connect Claude Desktop. That's it.

① Download the app

macOS    · coming soon
Linux    · coming soon
Get the installer →

⚠ On first launch Windows shows a SmartScreen warning because the app isn't code-signed yet. Click More info → Run anyway.

② Connect Claude Desktop

Open NeuroVault → Settings → Connect Claude Desktop
→ click "Copy JSON" → paste into Claude Desktop config

Works identically for Cursor, Codex, Continue, or any MCP-compatible agent. Each one gets the same knowledge graph.
